# QtCharts绘制动态心电图

## 简介
本项目提供了一个使用QtCharts库绘制动态心电图的示例。通过Qt的定时器功能，实现了心电图数据从左往右动态绘制的效果。当数据到达坐标轴的最大值后，会自动从左往右继续绘制，形成连续的动态心电图。

## 功能特点
- **动态绘制**：心电图数据会随着时间从左往右动态绘制。
- **循环绘制**：当数据到达坐标轴的最大值后，会自动从左往右继续绘制，形成连续的动态效果。
- **易于扩展**：代码结构清晰，便于根据需求进行扩展和修改。
